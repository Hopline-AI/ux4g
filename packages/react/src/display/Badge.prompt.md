**Badge** — compact status label (Approved, Pending, Rejected) or a count/dot indicator.

```jsx
<Badge variant="success">Approved</Badge>
<Badge variant="warning" appearance="tonal">Pending</Badge>
<Badge variant="danger" appearance="solid">Overdue</Badge>
<Badge variant="primary" appearance="dot" />
```

- **variant**: neutral · primary · success · danger · warning · info
- **appearance**: `tonal` (soft) · `solid` · `dot`
