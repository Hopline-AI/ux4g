**Select** — native dropdown matching the Input style. Provide `options` (strings or `{value,label}`) or `<option>` children. Requires the Material Symbols Rounded font for the chevron.

```jsx
<Select label="State / UT" placeholder="Select your state"
        options={["Andhra Pradesh", "Bihar", "Gujarat", "Karnataka"]} />
```

- **size**: `small` · `default` · `large` · **state**: `default` · `error` · `success`.
